## HOW TO UPLOAD TO GITHUB

1. Go to your `shes-the-goose` repository on GitHub
2. Click `Add file` → `Upload files`
3. Create a folder called `images` — type `images/` before each filename when uploading
4. Upload all photos with the exact filenames listed above
5. Commit the changes

Netlify will automatically rebuild the site within 1-2 minutes of each commit.
